# Blog

Personal portfolio and blog for Nikita Pochaev (@po4yka).

## Tech Stack

- [Vite](https://vite.dev/) 6 + [React](https://react.dev/) 18
- [Tailwind CSS](https://tailwindcss.com/) v4
- [shadcn/ui](https://ui.shadcn.com/) (Radix UI primitives)
- [React Router](https://reactrouter.com/) 7

## Getting Started

```sh
npm install
npm run dev
```

## Project Structure

```
src/
  app/
    components/   # UI components, data, stores
    pages/        # Route pages (Home, Blog, Projects, Experience, Settings, Admin)
    routes.ts     # React Router configuration
  styles/         # Tailwind, theme, fonts
docs/
  Guidelines.md   # Design system guidelines
  design/         # Figma design prompts (reference)
```
